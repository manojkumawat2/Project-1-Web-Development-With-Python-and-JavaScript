API Documentation

It is very easy to use API(Application Programming Interface) our website. You can register on our website and get the required url inorder to access to api of our website.
You can access the information of books with the help of following url with the required ISBN number of the book in JSON(Javascript Object Notation) format.

http://127.0.0.1:5000/api/isbn
JSON Format
{
"author": "Oscar Wilde",
"average_score": "None",
"isbn": "0140621733",
"review_count": 0,
"title": "Lady Windermere's Fan",
"year": 1893
}
